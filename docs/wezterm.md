# 💻 WezTerm ターミナルエミュレータ設定・キーバインド詳細

このドキュメントでは，[modules/apps/wezterm/](../modules/apps/wezterm/) に定義されている WezTerm ターミナルの機能設計，外観カスタマイズ，およびキーバインドについて詳細に解説します．

---

## 🎨 美しい外観と機能設計 (Design & Aesthetics)

配色は Matugen パレット (壁紙連動．[matugen-palette.md](matugen-palette.md) 参照) に追従します．

* **HackGen フォントの統合**: 日本語対応等幅フォント「`HackGen Console NF`」を指定しています．
* **85% 透過**: `window_background_opacity = 0.85` で本体を半透明化し，壁紙が透けます．
* **メイン領域と溶け込むタブバー**:
  - fancy タブバーはウィンドウ透過の外側で描画されアルファが黒に潰れるため，本体と同レイヤーの**レトロタブバー**を使用．バー地を本体と同じ「黒 0.85」で塗ることで境目なく馴染ませています．
  - タブは **平行四辺形** (左下三角 + 本体 + 右上三角)．アクティブ = accent，非アクティブ = surface グレー (nvim のタブラインと同じデザイン)．
* **ミニマルな1枚タブ非表示**: タブが1つの時はタブバーを自動で非表示にします．
* **余白**: 左右 1cell / 上下 6px．本文はタブバー直下に固定し，セル高の端数は下端に逃がしています．

## 🔍 インテリジェントなプロセス自動検知（動的タブタイトル）

`format-tab-title` イベントに，現在実行中のプログラム名（プロセス）を監視してタブタイトルをインテリジェントに上書きするカスタムスクリプトを組み込んでいます．

| 実行中のプロセス | 表示されるタブタイトル | 説明 |
| :--- | :--- | :--- |
| `cmd.exe` | **`CMD`** | Windowsコマンドプロンプト実行時 |
| `powershell.exe` / `pwsh.exe` | **`PowerShell`** | PowerShell起動時 |
| `wsl.exe` / `wslhost.exe` | **`Ubuntu`** | WSL (Ubuntu) 上の作業時 |
| `nvim` | **`Neovim`** | エディタ作業時 |
| その他 | *(実行中のバッファ名等)* | 標準タイトルをスマートにフォールバック |

また，タブのアクティブ状態を視覚的に強調するため，アクティブタブの両端には美しい三角形（Nerd Font `ple_lower_right_triangle` / `ple_upper_left_triangle`）のカラー枠が表示されます．

## ⌨️ キーバインド・操作ショートカット (Leader: Ctrl+Space)

WezTermの標準ショートカットをすべて無効化（`disable_default_key_bindings = true`）し，**独自の Leader キー操作系**（tmux/screen風）で完全に制御します．

* **Leader キー**: **`Ctrl + Space`** (入力後，2秒間入力待ち受け)

### 1. タブ操作 (Tab Management)

| キーバインド | mods | 動作内容 |
| :---: | :--- | :--- |
| **`t`** | `LEADER` | 新しいタブを作成 (デフォルト: WSL，カレントディレクトリ維持) |
| **`T`** | `LEADER + Shift` | PowerShell を新しいタブで開く (Windows連携用) |
| **`w`** | `LEADER` | 現在のタブを閉じる (確認ダイアログあり) |
| **`n`** | `LEADER` | 次のタブへ移動 (`Next`) |
| **`p`** | `LEADER` | 前のタブへ移動 (`Previous`) |
| **`{`** | `LEADER + Shift` | 現在のタブを左へ移動（並び替え） |
| **`}`** | `LEADER + Shift` | 現在のタブを右へ移動（並び替え） |
| **`[1-9]`** | `ALT` | 指定した番号のタブへ直接ジャンプ（Alt+1 で1枚目のタブへ） |

### 2. ペイン分割・移動 (Pane Management)

1つのタブ内を分割して効率的に作業するためのショートカットです．

| キーバインド | mods | 動作内容 |
| :---: | :--- | :--- |
| **`d`** | `LEADER` | ペインを **縦分割** (下に新規作成) |
| **`r`** | `LEADER` | ペインを **横分割** (右に新規作成) |
| **`x`** | `LEADER` | 現在のペインを閉じる (確認ダイアログあり) |
| **`z`** | `LEADER` | 現在のペインを一時的に最大化 / 復元 (Zoom) |
| **`h`** / **`j`** / **`k`** / **`l`** | `LEADER` | 分割されたペイン間を移動 (左 / 下 / 上 / 右) |
| **`a`** | `LEADER` | **ペインアクティブ化モード**に移行 (クイック切り替え) |
| **`s`** | `LEADER` | **ペインのリサイズモード**に移行 |

### 3. 特殊操作・ユーティリティ (Advanced Controls)

| キーバインド | mods | 動作内容 |
| :---: | :--- | :--- |
| **`[`** | `LEADER` | **コピーモード**の起動 |
| **`W`** | `LEADER + Shift` | **ワークスペース選択ランチャー** の起動 |
| **`$`** | `LEADER` | 現在のワークスペース名の変更プロンプトを表示 |
| **`p`** | `CTRL + Shift` | **WezTerm コマンドパレット** の起動 (あらゆるコマンドの検索・実行) |
| **`c`** | `CTRL + Shift` | クリップボードへコピー |
| **`v`** | `CTRL + Shift` | クリップボードから貼り付け |
| **`+`** / **`-`** / **`0`** | `CTRL` | フォントサイズの拡大 / 縮小 / リセット |
| **`r`** | `CTRL + Shift` | **WezTerm設定ファイルの強制リロード** (変更を即時反映) |
| **`m`** | `LEADER` | 自身の dotfiles リポジトリのキーバインド解説をブラウザで開く |

## 🛠️ 各種特殊モードの操作詳細

### A. ペインのリサイズモード (`Leader + s`)

リサイズモードに移行すると，以下のキーを入力することで，ペインの境界線を細かく動かすことができます．**`Escape`** または **`Enter`** でリサイズモードを終了します．

| キー | 動作内容 |
| :---: | :--- |
| **`h`** | 左へ境界線を動かす (縮小/拡大) |
| **`j`** | 下へ境界線を動かす (縮小/拡大) |
| **`k`** | 上へ境界線を動かす (縮小/拡大) |
| **`l`** | 右へ境界線を動かす (縮小/拡大) |
| **`Escape`** / **`Enter`** | リサイズモードを終了 |

### B. ペインのアクティブ化モード (`Leader + a`)

分割されたペインをキー単体で素早く選択してアクティブにするためのモードです．

| キー | 動作内容 |
| :---: | :--- |
| **`h`** | 左のペインを選択 |
| **`j`** | 下のペインを選択 |
| **`k`** | 上のペインを選択 |
| **`l`** | 右のペインを選択 |

### C. コピーモード (`Leader + [`)

ターミナルのログ出力をスクロールして遡り，Vimライクなキーバインドでテキストを範囲選択・コピーするための強力なモードです．

| キー | 動作内容 |
| :---: | :--- |
| **`Esc`** / **`q`** / **`Ctrl + c`** | コピーモードを終了 |
| **`h`** / **`j`** / **`k`** / **`l`** | カーソル移動 (左 / 下 / 上 / 右) |
| **`v`** | ビジュアルモード (文字単位選択の開始) |
| **`V`** (`Shift + v`) | ビジュアルラインモード (行単位選択の開始) |
| **`Ctrl + v`** | ビジュアルブロックモード (矩形選択の開始) |
| **`y`** | 選択範囲をシステムクリップボードにコピーしてコピーモードを終了 |
