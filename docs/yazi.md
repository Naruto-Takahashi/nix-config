# 📁 Yazi — CUIファイルマネージャ 設定・使い方詳細

Yaziは**Rust製の高速CUIファイルマネージャ**です．  
非同期I/Oと非同期プレビューによる極めて高いレスポンス性が特長で，Vim風のキーバインドを採用しています．  
本環境では **Cyberdreamテーマ** をベースに，背景透過・塗りつぶしフォルダアイコン・選択行ハイライトをカスタマイズして適用しています．

---

## 🚀 起動方法

```bash
yazi          # 通常起動（カレントディレクトリ）
yazi <path>   # 指定ディレクトリで起動
```

> [!TIP]
> **シェル統合（`ya` ラッパー）**: `enableZshIntegration = true` により，Yazi終了時に**シェルのカレントディレクトリが自動的に移動先に変わります**．  
> `cd` の代わりに `yazi` を使うことで，Yaziで移動した場所からそのまま作業を続けられます．

## 🗺️ 基本的な画面構成

```
┌─────────────────────────────────────────────────────┐
│  [親ディレクトリ]  │  [現在のディレクトリ]  │  [プレビュー]  │
│  (左ペイン)        │  (中央ペイン・メイン)   │  (右ペイン)    │
└──────────────────────────────────────────────────────┘
│  [モード表示]  [パーミッション]  [ファイル位置/総数]  [時刻]  │
└─────────────────────────────────────────────────────┘
```

- **左ペイン**: 親ディレクトリの内容（移動の手がかり）
- **中央ペイン**: 現在のディレクトリ（主な操作対象）
- **右ペイン**: 選択中ファイルのプレビュー（テキスト/画像/動画等に対応）

## ⌨️ キーバインド一覧

### 移動・ナビゲーション

| キー | 動作 |
|------|------|
| `j` / `↓` | 1つ下に移動 |
| `k` / `↑` | 1つ上に移動 |
| `h` / `←` | 親ディレクトリへ移動 |
| `l` / `→` / `Enter` | ディレクトリに入る / ファイルを開く |
| `gg` | リスト先頭へジャンプ |
| `G` | リスト末尾へジャンプ |
| `<C-u>` | 半ページ上スクロール |
| `<C-d>` | 半ページ下スクロール |
| `~` / `H` | ホームディレクトリへ移動 |
| `%` | カレントワーキングディレクトリへ移動（CWD） |
| `[` | 前のタブへ |
| `]` | 次のタブへ |

### ファイル操作

| キー | 動作 |
|------|------|
| `o` / `Enter` | ファイルを開く（規定アプリ） |
| `O` | ファイルを開くアプリを選択 |
| `y` | ファイルをコピー（ヤンク） |
| `x` | ファイルをカット |
| `p` | ファイルをペースト |
| `P` | 強制上書きペースト |
| `d` | ゴミ箱へ送る |
| `D` | 完全削除（確認あり） |
| `a` | 新規ファイル作成（末尾に `/` でディレクトリ作成） |
| `r` | ファイル名を変更（リネーム） |
| `;` | シェルコマンドを実行 |
| `!` | インタラクティブシェルを起動 |

> [!NOTE]
> 本環境では `o` キー（開く）に WSL対応の `wsl-open` を使用しています．  
> WSL環境では `wsl-open` が利用でき，Windows側の規定アプリでファイルを開けます．  
> デスクトップ環境では `xdg-open` にフォールバックします．

### 選択・マーキング

| キー | 動作 |
|------|------|
| `Space` | カーソル行をトグル選択 |
| `v` | ビジュアル選択モード（範囲選択） |
| `V` | 逆選択（選択反転） |
| `<Esc>` | 選択を解除 |
| `u` | 操作をUndo |

### 検索・フィルター

| キー | 動作 |
|------|------|
| `/` | ファイル名で検索（前方） |
| `?` | ファイル名で検索（後方） |
| `n` | 次の検索結果へ |
| `N` | 前の検索結果へ |
| `f` | フィルター（絞り込み表示） |
| `F` | グローバル検索（fd連携） |

### ソート・表示切替

| キー | 動作 |
|------|------|
| `,m` | 更新日時でソート |
| `,M` | 更新日時でソート（逆順） |
| `,s` | ファイルサイズでソート |
| `,S` | ファイルサイズでソート（逆順） |
| `,e` | 拡張子でソート |
| `,r` | ランダムソート |
| `.` | 隠しファイルの表示・非表示トグル |

### タブ管理

| キー | 動作 |
|------|------|
| `t` | 新しいタブを開く |
| `1`〜`9` | 指定番号のタブに切り替え |
| `[` | 前のタブへ移動 |
| `]` | 次のタブへ移動 |

### コピー・クリップボード連携

| キー | 動作 |
|------|------|
| `yy` | ファイルをコピー（ヤンク） |
| `yn` | ファイル名をシステムクリップボードにコピー |
| `yp` | ファイルパス（絶対）をクリップボードにコピー |
| `yd` | ディレクトリパスをクリップボードにコピー |

### その他

| キー | 動作 |
|------|------|
| `q` | Yaziを終了 |
| `Q` | Yaziを終了（シェルのCDを更新しない） |
| `?` | キーバインド一覧（ヘルプ）を表示 |
| `~` | テーマプレビューを表示 |
| `W` | 操作ログパネルを表示 |
| `J` | タスクマネージャーを開く |

## 🎨 テーマカスタマイズ（本環境の設定）

設定ファイル: [`modules/yazi.nix`](../modules/yazi.nix)

### 色彩設定（Cyberdreamテーマ準拠）

| 要素 | 色コード | 説明 |
|------|---------|------|
| ディレクトリアイコン・テキスト | `#5ea1ff` | cyberdream `blue`（nvimの `Directory` ハイライトと統一） |
| 選択行背景（current） | `#3c4048` | cyberdream `bg_highlight`（nvimの `CursorLine` と統一） |
| 親ディレクトリ行背景（parent） | `#3c4048` | 同上 |
| CWD表示 | `#5ef1ff` | cyberdream `cyan` |
| パーミッション区切り | `#5ea1ff` | cyberdream `blue` |
| 実行権 | `#5eff6c` | cyberdream `green` |
| 書き込み権 | `#ff6e5e` | cyberdream `red` |

### 透過設定

```toml
[app]
overall = { bg = "reset" }   # アプリ全体の背景透過

[mode]
normal_main = { fg = "#5ea1ff", bg = "reset" }   # モード表示も透過

[status]
percentage = { fg = "#ffffff", bg = "reset" }    # 「Top」「4K」等も透過
position   = { fg = "#ffffff", bg = "reset" }
```

### アイコン設定

```toml
[icon]
prepend_conds = [
    { if = "dir", text = "󰉋", fg = "#5ea1ff" }   # 塗りつぶしフォルダアイコン
]
prepend_dirs = [
    { name = ".config", text = "󰉋", fg = "#ff9800" },
    { name = ".git",    text = "󰉋", fg = "#00bcd4" }
]
```

> [!NOTE]
> アイコンの表示には **Nerd Fonts** 対応フォント（本環境では HackGen NF）が必要です．

### ステータスバーのセパレータ

```toml
[status]
sep_left  = { open = "", close = "" }   # 左セパレータ（なし）
sep_right = { open = "", close = "" }   # 右セパレータ（山括弧 < 形状）
```

## 💡 実践的なワークフロー例

### ファイルを素早くコピーして別ディレクトリに貼り付ける

```
1. コピーしたいファイルにカーソルを合わせる
2. yy でコピー（ヤンク）
3. l でディレクトリを移動（または h で上に戻る）
4. 目的のディレクトリで p でペースト
```

### 複数ファイルをまとめて削除する

```
1. v でビジュアル選択モードに入る
2. j / k で選択範囲を広げる（または Space で個別トグル）
3. d でゴミ箱へ（D で完全削除）
```

### シェルと連携してそのまま作業を続ける

```bash
# yazi を起動し，目的のディレクトリへ移動
yazi

# Yazi 内でディレクトリを移動してから q で終了
# → シェルが自動的に最後に開いていたディレクトリに移動している
pwd   # 確認
```

### プロジェクトのルートを素早く開く

```bash
# ghq 管理下のリポジトリをホームから直接 Yazi で開く
yazi ~/ghq/github.com/Naruto-Takahashi/
```
