# 🗔 i3wm (i3 window manager) キーバインド・設定詳細

このドキュメントでは、[modules/i3.nix](../modules/i3.nix) に定義されている i3 Window Manager の詳細なキーバインドおよび挙動設計について解説します。

---

## 🔑 基本操作 (Core Operations)

| キーバインド | 動作概要 | 備考 |
| :--- | :--- | :--- |
| `Super + Enter` | ターミナル (**WezTerm**) の起動 | Nvidia用GPUラッパー `nixGL` を自動適用 |
| `Super + Space` | アプリケーションランチャー (**Rofi**) の起動 | ※ `ALT + SPACE`（Kanata経由）からも起動可能 |
| `Super + Shift + Q` | 現在フォーカスされているウィンドウを閉じる | GNOME Forge とキーマップを統一 |

---

## 🗔 ウィンドウ操作・フォーカス (Window & Focus)

Vim ユーザーに馴染み深い **HJKL** 配列で直感的にウィンドウ操作が可能です。

### 1. フォーカスの移動
* `Super + H` ➔ 左のウィンドウへフォーカス
* `Super + J` ➔ 下のウィンドウへフォーカス
* `Super + K` ➔ 上のウィンドウへフォーカス
* `Super + L` ➔ 右のウィンドウへフォーカス

### 2. ウィンドウの位置移動
* `Super + Shift + H` ➔ ウィンドウを左へ移動
* `Super + Shift + J` ➔ ウィンドウを下へ移動
* `Super + Shift + K` ➔ ウィンドウを上へ移動
* `Super + Shift + L` ➔ ウィンドウを右へ移動

### 3. 表示レイアウト
* `Super + V` ➔ ウィンドウの分割方向の切り替え（縦分割 ⇄ 横分割）
* `Super + F` ➔ フルスクリーンの切り替え
* `Super + T` ➔ **タイリング / 浮動表示（Floating）のトグル切り替え** (※ `ALT + T` でも発動)

## 📥 最小化・復元 (Scratchpad Minimization)

i3wm 標準の Scratchpad（一時退避領域）を応用し、Windows/Mac のような最小化・復元体験を再現しています。

* **`Super + M` (最小化)**: アクティブなウィンドウを Scratchpad に退避させ、非表示にします（※ `ALT + M` でも発動）。
* **`ALT + TAB` / `Super + Shift + M` (復元)**:
  - 視覚的な一覧（Rofi Window Switcher）が起動します。
  - 最小化されていたウィンドウを選択すると、**独自の自動復元ラッパースクリプト (`rofi_window_wrapper.py`) が介入し、自動的にタイリング表示へと戻した上でデスクトップに復元します**（復元時に毎回浮動表示になってしまう i3wm の欠点を完全解消）。

## 📐 即時サイズ調整 (Direct Resizing)

リサイズモードに入ることなく、GlazeWMのようにキー単体でフォーカスウィンドウのサイズをダイレクトに拡大・縮小できます。

* `Super + U` ➔ ウィンドウ幅を縮小
* `Super + P` ➔ ウィンドウ幅を拡大
* `Super + I` ➔ ウィンドウ高さを縮小
* `Super + O` ➔ ウィンドウ高さを拡大

---

## 🎨 スクリーンショット機能 (Premium Screenshots)

超軽量スクリーンショットツール `maim` と `xclip` を組み合わせ、ゴールドの美しい半透明ドラッグ枠で直感的にキャプチャ可能です。

* **範囲選択スクリーンショット (クリップボード保存)**:
  - キー: `Super + Shift + S` または `Print`
  - 挙動: マウスでドラッグした領域（枠色はゴールド `#ffc20d`）をキャプチャし、クリップボードにPNG画像として保存します。チャットや各種ソフトに `Ctrl + V` で即時貼り付け可能です。
* **全画面スクリーンショット (ファイル保存)**:
  - キー: `Shift + Print`
  - 挙動: 画面全体のキャプチャを撮影し、`~/Pictures/` ディレクトリ配下に `Screenshot_日付_時間.png` として自動で整理保存します。

## 🌐 ワークスペース操作 (Workspace Controls)

ワークスペース間をスムーズに行き来し、ウィンドウを別ワークスペースへ瞬時に移動・追従できます。

* `Super + S` ➔ 次のワークスペースへ切り替え (※ `ALT + S` でも発動)
* `Super + A` ➔ 前のワークスペースへ切り替え (※ `ALT + A` でも発動)
* `Super + ` ` ` (バッククォート) ➔ 直前までアクティブだったワークスペースとの相互往復 (※ `ALT + ` ` ` でも発動)
* `Super + Shift + [1~0]` ➔ 現在のウィンドウを選択した番号のワークスペースへ移動し、自身もそのワークスペースへ自動追従します (Move and Follow)。

## 📊 ステータスバー (i3bar)

上部に配置されたステータスバーは、[i3status_wrapper.py](../modules/i3status_wrapper.py) を介してリアルタイムにシステムステータスを日本語 Mozambique スタイルで美しくレンダリングします。
* **メモリ使用量 (RAM)**: 常時監視。10%以下で「RAM MEMORY LOW」と赤色の警告出力。
* **CPU使用率**: 全体負荷をリアルタイムパーセンテージ監視。
* **Ethernet IP/速度**: 接続状況と有線ネットワーク速度を表示。
* **システム時刻**: `年-月-日 時:分:秒` を常時アップデート表示。
